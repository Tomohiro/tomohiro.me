tomohiro.me
================================================================================

My domain's records configuration with DNSimple by Terraform


Requirements
--------------------------------------------------------------------------------

- DNSimple token
- Atlas token
- Terraform


Getting Started
--------------------------------------------------------------------------------

Add tokens to `.envrc`:

```sh
$ cp .envrc.example .envrc
$ vi .envrc
```

Check the plan:

```sh
$ terraform plan
```
